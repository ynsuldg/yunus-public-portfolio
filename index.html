Projekt

Projekt

Webshop med mikrotjänster

Backend till en webshop byggd med mikrotjänstarkitektur. Systemet består av Auth-server, Product-service, Order-service och Email-service. Kommunikation sker via REST och RabbitMQ, JWT används för autentisering och PostgreSQL används för lagring.

Mitt huvudansvar var Order-service där jag arbetade med orderlogik, kommunikation mot Product-service, JWT-vidarebefordran och RabbitMQ-integration.

Java Spring Boot REST API JWT RabbitMQ PostgreSQL Docker Render

Two-Factor Authentication System

Säker autentiseringsapplikation byggd med Spring Boot och Spring Security. Användare kan registrera sig, aktivera tvåfaktorsautentisering och logga in med både lösenord och tidsbaserad engångskod från Microsoft Authenticator eller Google Authenticator.

Projektet använder PostgreSQL för lagring av användare och hemliga nycklar. QR-koder genereras automatiskt för enkel koppling mot authenticator-appar och åtkomst till skyddade sidor kräver både inloggning och verifierad MFA.

Java Spring Boot Spring Security PostgreSQL 2FA MFA Thymeleaf

Secure Webshop

Webbshop byggd med Spring Boot där fokus låg på säkerhet, användarhantering och GDPR-anpassning. Systemet innehåller registrering, inloggning, profilhantering, kundvagn, orderhantering och administratörsfunktioner.

Säkerheten implementerades med Spring Security, BCrypt, sessionshantering och CSRF-skydd. Projektet distribuerades med Docker till Render och använder PostgreSQL för lagring av användare, produkter och ordrar.

Java Spring Boot Spring Security PostgreSQL Docker Render GDPR Thymeleaf
